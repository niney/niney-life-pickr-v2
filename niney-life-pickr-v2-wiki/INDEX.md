# niney-life-pickr-v2 Knowledge Base

Last compiled: 2026-05-31
Total topics: 16 | Total concepts: 8 | Mode: codebase

선택을 대신 골라주는 서비스 — pnpm + Turborepo 모노레포(Fastify API + Vite 웹 + Expo 앱)의 컴파일된 위키. 처음 본다면 [project-overview](topics/project-overview.md) → 관심 토픽 순서로 읽는 것을 권장. 사용자 대상 공개 페이지(`/`, `/restaurants`)와 어드민 운영 도구(`/admin/restaurants`, `/admin/discover`, `/admin/auto-discover` 등) + 정산하기(`/restaurants/:placeId/settle/new` N차 wizard, `/me/settlements`, `/me/contacts`, `/share/settlements/:token`) 가 한 SPA 안에 분리되어 있고, 앱(`apps/mobile`)에도 정산 도메인 풀 구현 + Universal Links/App Links 로 공유 토큰 deep link 가 들어왔다.

> **용어**: 이 위키는 **웹**(`apps/web`) / **앱**(`apps/mobile`) / **모바일**(=웹의 작은 화면)을 구분해서 쓴다. 슬러그·디렉터리·스크립트 식별자는 `mobile`/`web` 그대로. 자세한 규칙은 [schema.md Terminology](schema.md#terminology--웹--앱--모바일).

## How to navigate

- **새 작업 시작** → [project-overview](topics/project-overview.md)에서 디렉터리 지도와 워크플로 파악
- **정산하기** (`/restaurants/:placeId/settle/new` N차 wizard / `/me/settlements` 이력 + 이어 입력 / `/me/contacts` 단골 관리 / `/share/settlements/:token` 비인증 공유 보기 + app deep link) → [settlement](topics/settlement.md) (도메인 전체 — N차 rounds · extraction · session · contact · share · draft · edit)
- **모바일 정산** (앱에 풀 구현, web 과 동일 4단계 UX + bottom-sheet 변형) → [mobile](topics/mobile.md) `app/restaurant/[placeId]/settle/`, `app/settlement/`, `app/share/settlements/[token].tsx`, `src/components/settlement/*`
- **공유 링크 deep link** (`/share/settlements/:token` 가 설치된 앱으로 가로채짐) → [mobile](topics/mobile.md) Universal Links/App Links + [friendly](topics/friendly.md) `well-known` 모듈 (env-driven AASA/assetlinks). 미설치 단말은 web SPA fallback.
- **백엔드 작업** → [friendly](topics/friendly.md), 크롤링 관련이면 [crawl](topics/crawl.md), LLM 관련이면 [ai](topics/ai.md), 지도 키 관리면 [map](topics/map.md), 정산이면 [settlement](topics/settlement.md)
- **어드민 자동 발견** (`/admin/auto-discover`) → [auto-discover](topics/auto-discover.md) → [crawl](topics/crawl.md) → [shared](topics/shared.md) → [web](topics/web.md)
- **어드민 발견 페이지** (`/admin/discover`) → [web](topics/web.md) → [crawl](topics/crawl.md) → [api-contract](topics/api-contract.md) → [shared](topics/shared.md)
- **공개 사용자 페이지** (`/`, `/restaurants`) → [web](topics/web.md) → [shared](topics/shared.md) → [friendly](topics/friendly.md) → [map](topics/map.md)
